# 💬 Prompt de Início — Digital Monk Marketing Hub

Cole o prompt abaixo no chat do Antigravity para ativar o agente de marketing.

---

## ▶️ Prompt Padrão

```
Leia o AGENT.md deste projeto e me confirme quais ferramentas MCP estão disponíveis. Estou pronto para planejar uma nova campanha.
```

---

## 🎯 Exemplos de Comandos no Chat

Após ativar o agente, você pode pedir:

**Planejar uma campanha:**
```
Planeje uma cadência de 4 emails para o setor de telecomunicações usando os insights do NotebookLM Glassbox.
```

**Buscar contatos:**
```
Liste os contatos do Bigin com a tag "telecom" que ainda não receberam nenhum email.
```

**Criar e enviar um email:**
```
Crie o Email 1 de descoberta personalizado para [Nome] da [Empresa] e envie pelo Bigin.
```

**Gerar conteúdo:**
```
Com base no NotebookLM Glassbox, escreva um artigo LinkedIn sobre como times de CX podem reduzir o MTTR em 2026.
```

**Registrar progresso:**
```
Registre no Bigin que enviamos o Email 1 da cadência Glassbox para todos os contatos da tag "telecom" hoje.
```

---

> O agente lê o `AGENT.md`, usa o Bigin CRM via MCP para acessar contatos e enviar emails,
> e consulta o NotebookLM para gerar insights e conteúdo baseado em dados reais.
